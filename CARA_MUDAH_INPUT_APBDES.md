## LANGKAH 2: CEK DATA BIDANG (HANYA SEKALI SAJA)

## LANGKAH 3: BUAT LAPORAN TAHUN 2024

1. Di menu **"APBDes"**, klik **"Laporan APBDes"**
2. Klik tombol **"Create"** (biasanya warna biru)
3. Isi form seperti ini:
    ```
    Tahun: Pilih "2024"
    Nama Laporan: Ketik "APBDes Desa [Nama Desa Anda] 2024"
    Bulan Rilis: Pilih bulan sekarang
    Status: Pilih "Draft"
    ```
4. Klik **"Create"** untuk menyimpan
5. **Ingat nama laporan ini!** Nanti akan dipilih lagi

---

## LANGKAH 4: INPUT DATA PENDAPATAN

1. Di menu **"APBDes"**, klik **"Input Anggaran"**
   ğŸ’¡ _Catatan: Hanya ada menu "Input Anggaran", menu "Detail APBDes" sudah disembunyikan_
2. Klik tombol **"Create"**
3. Isi form pendapatan pertama:
    ```
    Laporan APBDes: Pilih "APBDes Desa [Nama Desa] 2024"
    Bidang APBDes: Pilih "Pendapatan Desa"
    Uraian: Ketik "Pendapatan Asli Desa"
    Anggaran: Ketik angka tanpa titik, contoh: 50000000
    Realisasi: Ketik angka tanpa titik, contoh: 45000000
    Bulan Realisasi: Pilih bulan data
    ```

5. **Ulangi langkah 2-4** untuk pendapatan lainnya:
    - Transfer Dana Desa

## LANGKAH 5: INPUT DATA BELANJA

1. Masih di menu **"Input Anggaran"**, klik **"Create"** lagi
2. Isi form belanja pertama:
    ```
    Laporan APBDes: Pilih "APBDes Desa [Nama Desa] 2024"
    Bidang APBDes: Pilih "Bidang Penyelenggaraan Pemerintahan Desa"
    Uraian: Ketik "Belanja Pegawai"
    Anggaran: Ketik angka, contoh: 200000000
    Realisasi: Ketik angka, contoh: 180000000
    ```
3. Klik **"Create"**
4. **Ulangi untuk semua belanja**, ganti bidangnya:
    - Bidang Penyelenggaraan Pemerintahan Desa
    - Bidang Pelaksanaan Pembangunan Desa
    - Bidang Pembinaan Kemasyarakatan
    - Bidang Pemberdayaan Masyarakat
    - Bidang Penanggulangan Bencana, Darurat dan Mendesak

---

## LANGKAH 6: CEK HASILNYA

1. Di menu **"APBDes"**, klik **"Dashboard APBDes"**
2. Lihat 4 kotak di atas:
    - Total Pendapatan (warna hijau)
    - Total Belanja (warna kuning)
    - Surplus/Defisit (warna biru/merah)
    - Laporan Aktif
3. **Kalau angkanya salah, kembali ke "Input Anggaran" untuk edit**

---

## LANGKAH 7: PUBLIKASIKAN (WAJIB!)

1. Kembali ke menu **"Laporan APBDes"**
2. Cari laporan yang tadi dibuat
3. Klik tombol **"Edit"** (biasanya ikon pensil)
4. Ubah **Status** dari "Draft" menjadi **"Diterbitkan"**
5. Klik **"Save"**
